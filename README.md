# CustomerBase

Android project built with <b>Jetpack Compose</b> following <b>Clean Architecture guideline</b>.</br>

### Current features:

- `Kotlin` with `Compose`
- `Hilt`
- `Navigation Component`
- `DataStore Preferences`
- `Room`
